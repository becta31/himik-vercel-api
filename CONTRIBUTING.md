
# 🤝 Как внести вклад

Спасибо, что хотите помочь проекту "ХИМИК"!

## 🐛 Нашли ошибку?

Если вы нашли ошибку (bug) в коде или на сайте:
1. Проверьте, нет ли уже созданной задачи (Issue) на эту тему.
2. Если нет, создайте новую **Issue**, опишите проблему как можно подробнее (шаги для воспроизведения, скриншоты).

## 💡 Предложение новой функции

У вас есть идея, как улучшить проект?
1. Создайте **Issue** с тегом `enhancement`.
2. Опишите, что вы хотите добавить и почему это будет полезно.

## 🔧 Хотите написать код?

1. Сделайте форк репозитория (Fork).
2. Создайте новую ветку (`git checkout -b feature/my-new-feature`).
3. Внесите изменения и зафиксируйте их (`git commit -m 'Add some feature'`).
4. Отправьте изменения в свой форк (`git push origin feature/my-new-feature`).
5. Создайте **Pull Request**.

Мы рассмотрим ваш код и, если всё в порядке, примем его!

## 📝 Стиль кода

*   **Frontend:** Соблюдайте отступы и комментарии. Используйте семантические теги HTML.
*   **Backend:** Следуйте стандартам ESLint для Node.js.
