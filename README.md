🚀 О проекте

"ХИМИК" — это интерактивная платформа для изучения химии. Проект создан как пример современного full-stack приложения на базе Serverless-архитектуры.

Ключевые фичи:

🔥 Интерактивная таблица: Изучайте элементы через красивые карточки.
⚡ Викторина: Проверяйте знания в режиме реального времени.
🏆 Рейтинг: Соревнуйтесь за место в топ-10.
📱 Адаптивность: Работает на любом устройстве.
🛠 Технологии

Frontend: HTML, Tailwind CSS, Vanilla JS
Backend: Node.js (Serverless Functions)
Database: MongoDB Atlas
Deploy: Vercel
📂 Структура

/ — Страницы приложения (Landing, Study, Quiz).
/api — Серверные функции для работы с БД.
elements.json — База данных химических элементов.
ARCHITECTURE.md — Подробное описание архитектуры проекта.
🏃 Как запустить локально

Так как проект использует загрузку данных (fetch), его нужно запускать через локальный сервер.

Клонируйте репозиторий:
git clone https://github.com/becta31/himik-vercel-api.git
 Откройте папку в VS Code.
Запустите через расширение Live Server
