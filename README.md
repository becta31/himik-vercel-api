## 🌳 Контроль версий с помощью Git (PyCharm / Терминал)

Для отслеживания изменений, совместной работы и публикации проекта (через GitHub Pages) используется система **Git**.

### 1\. Начало работы (Клонирование)

Если вы начинаете работу над проектом на новом компьютере:

1.  **Откройте** терминал или Git-интерфейс PyCharm.
2.  **Клонируйте** репозиторий с GitHub:
    ```bash
    git clone https://becta31.github.io/HIMIK/
    ```
3.  **Перейдите** в папку проекта: `cd HIMIK`

### 2\. Рабочий процесс обновления (После изменений)

Когда вы внесли изменения (например, исправили данные в `elements.json` или код в `study.html`):

| Шаг | Команда в терминале | Действие в PyCharm | Назначение |
| :--- | :--- | :--- | :--- |
| **1. Добавить** | `git add .` | Кнопка **Commit** -\> Выбрать файлы | Добавляет измененные файлы к фиксации. |
| **2. Фиксация** | `git commit -m "Описание изменений"` | Кнопка **Commit** -\> Ввести сообщение | Создает локальную точку сохранения. |
| **3. Отправить** | `git push origin main` | Кнопка **Push** | Отправляет изменения на GitHub. **Запускает автоматическое обновление сайта\!** |
| **4. Получить** | `git pull origin main` | Кнопка **Pull** (Обновить проект) | Загружает последние изменения с GitHub (если работали с другого устройства). |

### 3\. Работа с ветками (Рекомендуется для новых функций)

Для разработки новых функций, которые могут сломать работающий сайт, рекомендуется использовать отдельную ветку:

1.  **Создать новую ветку:** `git checkout -b new-feature-name`
2.  **Переключиться на основную:** `git checkout main`
