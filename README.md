README: HIMIK - Тренажер по химии
Тренажер по химии, разработанный с использованием HTML, Tailwind CSS и чистого JavaScript.
📁 Структура проекта
HIMIK/
├── index.html        # Главная страница, меню
├── study.html        # Режим обучения (периодическая таблица)
├── quiz.html         # Режим викторины
├── elements.json     # Данные о химических элементах (JSON)
└── README.md         # Этот файл
________________________________________
🛠️ Как запустить проект
У вас есть два основных способа запустить проект:
Способ 1: Прямое открытие в браузере (Рекомендуется, если работает)
Это самый простой способ, который работает в большинстве современных браузеров (особенно в Firefox и Edge).
1.	Найдите файл index.html в папке проекта.
2.	Дважды кликните по файлу index.html или откройте его через меню браузера ("Открыть файл").
Если вы увидите предупреждение в консоли разработчика или элементы не загрузятся в study.html (из-за ошибки CORS при чтении elements.json), переходите к Способу 2.
________________________________________
Способ 2: Запуск локального веб-сервера (Для обхода CORS)
Для стабильной работы с локальным файлом elements.json (особенно в браузере Google Chrome) рекомендуется использовать простой локальный сервер.
А. Использование Python (Самый простой вариант)
Если у вас установлен Python (версии 3.x), вы можете запустить сервер одной командой:
1.	Откройте командную строку (или терминал) в папке, где находится файл index.html.
2.	Выполните команду:
Bash
python -m http.server
3.	Откройте браузер и перейдите по адресу:
4.	http://localhost:8000/
Б. Использование Live Server (Для разработчиков)
Если вы используете Visual Studio Code (VS Code), это самый удобный вариант:
1.	Установите расширение Live Server (автор Ritwick Dey).
2.	Откройте файл index.html в VS Code.
3.	Кликните правой кнопкой мыши в редакторе и выберите "Open with Live Server" (или нажмите кнопку "Go Live" в правом нижнем углу).
Сервер автоматически откроет проект в браузере, и любые изменения в коде будут обновляться мгновенно.

🔧 Файлы для модификации
•	elements.json: Используется для изменения данных о химических элементах (например, добавление поля description, которое вы сделали, или исправление свойств).
•	study.html: Основной файл для работы с интерфейсом периодической таблицы и модальным окном.
•	quiz.html: Файл для логики викторины и ее вопросов.
________________________________________
💡 Полезная информация
•	Стилизация: Для стилизации проекта используется фреймворк Tailwind CSS (подключен через CDN в <head> каждого HTML-файла).
•	Данные: Все данные для обучения и викторины берутся из файла elements.json.
•	Навигация: Переключение между страницами (study.html, quiz.html) реализовано через ссылки в index.html.
